# 「Kling AI」API ドキュメント翻訳

## 1. Kling AI とは
Kling AI は、快手（Kuaishou）のラージモデル・アルゴリズムチームが独自に開発した画像生成「Kolors」と動画生成「Kling」技術を基盤とした、新世代の AI クリエイティブ生産ツール群です。AI 画像・AI 動画の生成に加え、多彩な編集コントロールを提供します。

- **クリエイター向け（個人・法人）**: Web／モバイル上で使えるオンライン制作プラットフォームとツールを提供。
- **開発者向け（個人・法人）**: 既存サービスへの組み込みを想定した API ソリューションを提供。

## 2. Kling AI の強み

### 2.1 強力な画像生成性能
- 「中国語圏で最も熟練した画像生成モデル」を標榜。LLM と数十億件の中国語コーパスで学習し、SDXL／SD3 といった OSS や Midjourney といったクローズドモデルを総合指標で上回り、中国語ユースケースで SOTA を達成。

| 機能 | 説明 | 生成の特長 |
| --- | --- | --- |
| Text to Image | テキストプロンプトから自由に画像を生成 | 多彩なスタイル、高品質な描写、創造性の高さ |
| Image to Image | 参照画像を入力しモデルの参照度を調節可能 | プロンプト一致性、リファレンス画像との整合性 |
| Multi-Image to Image | 複数画像の要素（被写体・シーン・スタイル）を融合 | マルチ画像融合、被写体の一貫性 |
| Image Expansion | 任意のアスペクト比や拡張領域を指定して生成 | ビューの拡張、拡張領域のカスタマイズ |

#### サンプルプロンプト
- 「水面に浮かぶ白髪の男性。ミニマルなスタイルで、広大なブルー、アニメ調、鮮やかな配色。トレンディで、深いホワイトとアンバーのトーン。ゆるい筆致で、神秘的かつディテール豊富。韓国風の美学。」
- 「city street（都市の街並み）」

### 2.2 ハイライト

#### ハイライト 1: 高度なセマンティック理解性能
- テキスト表現の継続的な改良と、画像テキストデータの精緻なアラインメントにより、高精度な意味把握を実現。
- 例: 「上海の街を歩くスタイリッシュな中国人女性。日差しが差し、黒いレザージャケットと赤いロングスカート、黒いバックパック、サングラス、赤い口紅。自信に満ちて軽やかに歩く。」

#### ハイライト 2: 映画的な画質テクスチャ
- 多段階の段階的学習、強化学習、人間の審美観に合わせた精密なアラインメントで、シネマティックな質感を実現。
- 例: 「カメラで撮影したようなシーン。群青色の湖面に二羽の白鳥が映り込み、遠景には連なる雪山。大きな絞り、ハイコントラスト、ハイサチュレーションで左右対称の構図を形成。」

#### ハイライト 3: 多条件下での生成制御
- 条件の組み合わせ生成をサポートし、人物の特徴保持やテキスト埋め込みなどコントロール性に優れる。
- 例: 「油絵調で描かれた上海・外灘。青空の下、モネ風の柔らかな筆致とモランディ配色で、光と影・鮮やかな色彩・豊かなディテールを備えた傑作。」
